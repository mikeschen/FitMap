# FitMap

An Android app that tracks the number of steps you take, calories eaten and calories burned.

#### Authors:  Virgilio Ramon de la Cruz, Polina Nenchev, Matt Rosanio, Michael Schennum

## Description

FitMap tracks the number of steps you take each day, as well as calories burned and calories consumed.

## Technologies

Android, Java, XML.

## Setup

Ensure you have Android Studio installed.

Clone this repository into your AndroidStudioProjects directory.

Use AVD Manager to run the project on a virtual device, or connect an Android phone with USB debugging enabled to run on a real device.
